# 📰 NewsWeaver - Website

A beautiful, minimal web interface to visualize your aggregated tech news with style.

## 🎨 Features

### Design Philosophy: Minimal & Modern
- **Clean Typography**: System fonts for fast loading and clarity
- **Whitespace**: Generous spacing for pleasant reading experience
- **Smooth Interactions**: Subtle hover effects and animations
- **Dark Theme Header**: Modern contrast with light content area
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile

### Interactive Features
✨ **Search**: Real-time search across article titles and URLs
📁 **Source Filters**: Filter articles by news source (Hacker News, Dev.to, etc.)
📊 **Statistics**: Live counts of total articles, unique sources, and last update time
🎯 **Time Indicators**: Shows when each article was published (e.g., "2h ago")
🔗 **Direct Links**: One-click access to original articles
⏰ **Auto-loading**: Data updates instantly when data.json changes

## 🚀 Quick Start

### Option 1: Open in Browser (Easiest)
1. Just double-click `index.html` to open in your default browser
2. The site will load your `data.json` automatically

### Option 2: Run Local Server (Recommended)
```bash
python server.py
```
This will:
- Start a local web server on `http://localhost:8080`
- Automatically open the site in your browser
- Handle file caching properly
- Press `Ctrl+C` to stop

### Option 3: Use Python's Built-in Server
```bash
# Python 3
python -m http.server 8080

# Then open http://localhost:8080 in your browser
```

## 📋 Requirements

- `data.json` in the same directory (generated by `scraper.py`)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies needed!

## 🎯 What You Can Do

1. **Search Articles**
   - Type in the search box to find articles by keyword
   - Search works on both title and URL

2. **Filter by Source**
   - Click "Filter by Source" button
   - Select sources you want to see
   - Multiple selections supported

3. **View Statistics**
   - Total articles in your dataset
   - Number of unique sources
   - Last time data was updated

4. **Read Articles**
   - Click "Read Article" to open the original article
   - Opens in a new tab without losing your place

## 🎨 Customization

To customize colors, edit the CSS variables at the top of `index.html`:

```css
:root {
    --primary: #1a1a1a;      /* Header background */
    --accent: #0066ff;        /* Links and highlights */
    --text-dark: #1a1a1a;    /* Main text */
    --text-light: #666666;   /* Secondary text */
    --border: #e5e5e5;       /* Border color */
}
```

## 🔄 Workflow

1. **Generate News**: Run your scraper
   ```bash
   python scraper.py
   ```

2. **View Website**: Run the server
   ```bash
   python server.py
   ```

3. **Browse Articles**: Open http://localhost:8080

4. **Repeat**: Scraper updates data.json, refresh browser to see new articles

## 🐛 Troubleshooting

**"Data file not found"**
- Make sure `data.json` is in the same directory as `index.html`
- Run `python scraper.py` first to generate data

**"Can't connect to server"**
- Make sure port 8080 is not in use
- Try a different port: `python -m http.server 8081`

**"Articles look broken"**
- Check browser console (F12) for errors
- Verify `data.json` has valid JSON format
